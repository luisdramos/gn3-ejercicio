API 

Departamentos 
-> El metodo GET Obtiene todos los departamentos Registrados 
-> El metodo GET + id Obtiene el departamento por ID 
-> El metodo POST Actualiza o Crear un Departamento
-> El metodo DELETE Elimina un departamento por ID
